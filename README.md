# Airbnb Experiences clone 🏡
In this project an Airbnb Experiences clone is built using React and other tools

Concepts used:
- [x] Custom components<br>
- [x] Reusable Components
- [x] React Props
- [x] array .map()
- [x] Conditional Rendering
- [x] Styling with Classes
<br>
